TeclaDesktop

TeclaDesktop is a client for making desktops TeclaShield Accessible which will allow users to generate Computer events through their Shields using a bluetooth network.

Developing TeclaDesktop:

1)TeclaDesktop uses Netbeans as its IDE.You can download it from here. http://netbeans.org

2)Clone the repository and checkout to branch desktop
"git checkout desktop"

3)Open the TeclaDesktop project in Netbeans by:
File-> Open Project -> ($Cloned directory)/TeclaShield/TeclaDesktop.
Select TeclaDesktop.A cup like icon should appear.Make sure you have the nbproject folder in TeclaDesktop/
